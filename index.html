Tecnologías

El servidor que hostea esta página está proveido por Microsoft, mediante Azure. Según el link que estés usando, ¡el servidor es o una de sus máquinas virtuales o una web estática!

Esta página es, de momento, completamente estática... Es decir, no recuerda tu historial, ni tiene manera de iniciar sesión e interactuar. Si eres aventurero, puede que hayas visto un archivo PHP vacío en el source de la web - tan solo se usa para configurar el host de Azure correctamente, ¡pero en realidad no lo utilizo para ninguna otra cosa!

El contenido y estilo han sido creados por vs133 ligeramente basados en la guía de interfaces Material de Google.

El source code de esta web es público en un repositorio de GitHub - te invito a que le eches un vistazo.
